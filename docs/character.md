# Character Crate Map

Use this index to jump into the major subsystems under `character/`. Each section links back to the shared creation/edit flows described in `docs/character_flow.md`.

## Subsections
- **Personal (person.rs)** – identity prompts, nickname/call-sign storage, and edit menus.
- **Civic (nation.rs)** – nationality, social class, and prestige-driven level-ups.
- **Physical (physical/)** – BMI logic, physique tiers, hand control/power stats, and weapon gating rules.
- **Vitality (vitality/)** – elemental choices + future spell hooks that the main crate mirrors.
- **Warrior (warrior/)** – combat classes, fighting styles, weapons, and how they sync with the new tier ladder.

_Add new sections here as we introduce auxiliary systems (companions, reputation, etc.)._
